# Exporting Data

## Exporting metadata to CSV

You can download a CSV file of the results by clicking the Export button and choosing "CSV".

TBD screenshot of Export > CSV

The CSV will include the metadata displayed in the search results in tabular format - for every record in the search results, not just the first page of results displayed.
